Pointers

Pointers are variables that stores memory address of another variables/objects.

Syntax :

int* ptr = &a;

Lets understand it with sample code

#include <iostream>
using namespace std;
int main(){
int a = 5;
int* ptr = &a;
cout<<ptr;
}

output : 0x2128

In code above, we initialized a variable of name a with value 5. just after that we initialized pointer variable by writing int* ptr with value &a. This is how we initialize pointer variable.

Note: The output will differ from compiler to compiler.

But before going into details about pointer, lets talk about these two & , * operators.

& Operator

Also known as Ampersand, when used before a variable, it returns memory address of that variable.

Here is example

#include <iostream>
using namespace std;
int main(){
int a = 5;
cout<<&a;
}

output : 0x2148

* Operator

Also known as Asterisk, it has three different operations, based on how it used.

For Multiplication

Here is example

#include <iostream>
using namespace std;
int main()
{
cout<<5*5;
}

output : 25

For declaring pointer variable.

By writing * after int or before variable name tells compiler that it is pointer variable.

Here is example

#include <iostream>
using namespace std;
int main(){
int* ptr = &a;
int *ptr = &a; //we can write * before variable name
cout<<ptr;
}

output : 0x21239

For derefencing pointer variable

Derefencing means extracting the value located at some address. When declaring pointer variable we use * to tell the compiler that it is pointer variable and not a regular one. Therefore pointer stores address of any variable, pointer is initialized with. Now, to get value stored at that address using pointer variable we use *var_name. Thus by derefencing pointer variable, value at that address is extracted.

Here is example

#include <iostream>
using namespace std;
int main(){
int a = 17;
int* ptr = &a;
cout<< *ptr <<" stored at address " << ptr ;

return 0;
}

output : 17 stored at address 0x23456

This is how it looks like in memory.

Variable Name Value Address
a 17 0x23456
ptr 0x23456 0x56749

Note: If we try to access *ptr in output without assigning it any variable address, it will throw segfault error.

Pointer Types

Pointer can store address of variables of different data types.

#include <iostream>
using namespace std;
int main(){

int a = 17;
int* ptr = &a; //int type

double b = 23.12;
double *ptr_b = &b; //double type

string s = "hello";
string *ptr_s = &s; //string type

cout<< *ptr <<" ;

cout<< *ptr_b <<" ;

cout<< *ptr_s <<" ;


return 0;
}

 Applications of Pointers

1. Changing the passed parameters.
2. Passing large objects.
3. Dynamic Memory Allocation
4. Implementing data structures like, linked list, tree, bst, etc.
5. To do System Level programming.
6. To return multiple values.
7. Used for accessing array elements.

Note : Disadvantage of reference over pointer is, reference cannot be redeclared.
