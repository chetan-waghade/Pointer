Pointers Introduction

- Pointers are variables that stores address of another variables.

Syntax :   int a=5; int* ptr = a;

Before understanding pointers we need to understand * operator. It is used for different purposes in c++.

For mulitplying numbers : int a=5,b=5,c; c=a*b;

For getting value from address : int a=5; cout<< *(&a); output: 5.

For creating pointer variable : int a=5; int* ptr=&a; cout<<ptr; outut: 0x2120.
