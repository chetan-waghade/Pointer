Pointers

Pointers are variables that stores memory address of another variables/objects.

Syntax :

int* ptr = &a;

Here is example code

#include <iostream>
using namespace std;
int main(){
int* ptr = &a;
cout<<ptr;
}

output : 0x2128

Note: The output will differ from compiler to compiler.

But before going into details about pointer, lets talk about these two & , * operators.

& Operator

Also known as Ampersand, when used before a variable, it returns memory address of that variable.

Here is example

#include <iostream>
using namespace std;
int main(){
int a= 5;
cout<<&a;
}

output : 0x2148

* Operator

Also known as Asterisk, it has three different operations, based on how it used.

For Multiplication

Here is example

#include <iostream>
using namespace std;
int main()
{
cout<<5*5;
}

output : 25

For declaring pointer variable.

By writing * after int or before variable name tells compiler that it is pointer variable.

Here is example

#include <iostream>
using namespace std;
int main(){
int* ptr = &a;
int *ptr = &a; //we can write * before variable name
cout<<ptr;
}

output : 0x21239

For derefencing pointer variable

Derefencing means extracting the value located at some address. When declaring pointer variable we use * to tell the compiler that it is pointer variable and not a regular one. Therefore pointer stores address of any variable, pointer is initialized to. Now, to get value stored at that address using pointer variable we use *var_name. Thus by derefencing pointer variable value at that address is extracted.

Here is example

#include <iostream>
using namespace std;
int main(){
int a = 17;
int* ptr = &a;
cout<< *ptr <<" stored at address " << ptr ;

return 0;
}

output : 17 stored at address 0x23456
